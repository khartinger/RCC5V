## 🏗️ Modulrahmen / Unterbau

| Deutsch | Englisch | Bemerkung / Dateiname |
|---------|---------|----------------------|
| Rahmen (Holz) | **Baseboard_Frame** | trägt Platte, Querstreben, Beine |
| Grundplatte | **Baseboard_Top** | die eigentliche Platte für Landschaft |
| Querstrebe | **Cross_Member** | horizontales Teil zur Stabilisierung |
| Diagonalstrebe / Verstrebung | **Diagonal_Brace** | für Aussteifung, oft schräg |
| Beine / Stützen | **Legs** | einfach, verständlich |
| Verbindungselemente zwischen Modulen | **Module_Connectors** | Kupplungen oder Schraubenleisten |

---

## 🛤️ Gleise / Trasse

| Deutsch | Englisch | Bemerkung / Dateiname |
|---------|---------|----------------------|
| Schotterbett | **Track_Ballast** | unter/um die Schwellen |
| Bahndamm | **Embankment** | aufgeschütteter Unterbau für Gleise |
| Querschnitt durch das Gleis | **Track_Cross_Section** | Zeichnungsbezeichnung oder Bilddatei |
| Weiche mit Unterflurantrieb | **Switch_with_Underfloor_Motor** | US-Englisch, modul-spezifisch |
| Unterflurantrieb | **Underfloor_Switch_Motor** | als Bauteilbezeichnung |
| Abdeckung Unterflurantrieb | **Underfloor_Switch_Motor_Cover** | klar & international |
| Prellbock | **Buffer_Stop** | UK/US international verständlich |

---

## 📦 Module & Layout

| Deutsch | Englisch | Bemerkung / Dateiname |
|---------|---------|----------------------|
| Module (einzelne Segmente) | **Module** | neutral, international |
| Modul-Anordnung / Anlage | **Module_Layout** | kurz, gut für Dateinamen |
| Modularanlage (gesamt) | **Modular_Model_Railway** | formal, Präsentation |
| Landschaftsdeckplatte / Moduloberfläche | **Baseboard_Top_Surface** | optional, wenn Fokus auf Gelände |

---

## 🔧 Hinweise
- **US vs. UK Unterschiede:**  
  - „switch“ (US) ↔ „point“ (UK)  
  - „ballast“ = international
